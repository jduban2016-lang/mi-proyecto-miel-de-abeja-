# 🐝 Proyecto Miel Dorada

Sitio web responsivo para el negocio de miel de abeja, desarrollado con HTML, CSS y JavaScript.

* **URL del proyecto:** https://tu-proyecto.vercel.app

---

## 🌐 APIs Utilizadas

1. **Indicadores Económicos (mindicador.cl):** Utilizada para mostrar el valor actualizado del dólar.
2. **Feriados en Chile:** Utilizada para desplegar los días festivos nacionales.
3. **OpenWeatherMap (Clima):** Utilizada para mostrar el estado del tiempo actual.

### 💡 Justificación del Tercer Dato (El Clima)
La integración de la API del clima en una tienda de miel es fundamental, ya que la apicultura depende al 100% de las condiciones climáticas. El clima afecta directamente la floración, el comportamiento de las abejas y las temporadas de cosecha de miel. Mostrar este dato aporta un valor educativo y contextual para el cliente interesado en el origen del producto.
